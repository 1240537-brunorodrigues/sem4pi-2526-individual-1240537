# US057 - Add an Engine Model to an Aircraft Model

## User Story

As a Backoffice Operator, I want to add an engine model to an aircraft model's list of certified engines.

Engine type must be compatible. The same engine model cannot be added twice to the same aircraft. An aircraft model might have several aircraft variants, corresponding to combinations of aircraft model and engine configuration.
