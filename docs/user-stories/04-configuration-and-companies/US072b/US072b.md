# US072b - List Fleet by Maker

## User Story

As an Air Transport Company Collaborator, I want to list my company's fleet by aircraft maker.

This listing should allow the collaborator to consult the aircraft in the company's fleet grouped or filtered by the aircraft model's maker/manufacturer.
