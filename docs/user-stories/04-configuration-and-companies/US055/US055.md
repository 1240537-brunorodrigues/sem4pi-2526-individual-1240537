# US055 - Create an Aircraft Model

## User Story

As a Backoffice Operator, I want to register a new aircraft model.

Any model must have at least one engine model. Aircraft has a model name and manufacturer, and their combination must be unique. At least one certified engine model must be associated. An aircraft model will have a type, maximum range and other flight characteristics. This must also be achieved by a bootstrap process.
