# US052 - Create an Airport

## User Story

As a Backoffice Operator, I want to register an airport in a given air control area.

An airport must be associated with exactly one air control area. The airport has an ICAO and IATA code that must be unique worldwide. An airport has location coordinates that must be valid and an elevation in meters above sea level. This must also be achieved by a bootstrap process.
