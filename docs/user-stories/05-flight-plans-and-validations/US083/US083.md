# US083 - Core Flight DSL

## User Story

As a Project Manager, I want the team to define and implement a Core Flight DSL to describe flight plans.

The Core Flight DSL must allow flight plans to be represented in files and validated through lexical, syntactic and semantic analysis before being imported into the system.
