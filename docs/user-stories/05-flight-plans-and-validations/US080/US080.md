# US080 - Create a Flight Plan

## User Story

As a Pilot, I want to register a flight plan for a route.

The flight plan must include the aircraft, departure date/time, fuel quantity and pilot. The pilot must be of the route's company. The flight plan status is set to "draft" when created and must undergo a multi-step validation process.
