# User Stories Documentation

This folder contains the documentation for the project user stories.

User stories are organized by functional area.

## Documentation Levels

Some user stories represent project management or infrastructure concerns and are documented using a simplified format.

Functional user stories are documented with:

1. Requirements Engineering
2. Analysis
3. Design
4. Implementation and Testing

## Areas

- Project Management
- Identity and Backoffice
- Weather Service
- Configuration and Air Transport Companies
- Flight Plans and Validations
- Simulation and Control
