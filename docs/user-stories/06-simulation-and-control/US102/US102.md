# US102 - Detect Aircraft Safety Violations in Real Time

## User Story

As a simulation system, I want to continuously monitor aircraft positions for overlaps so that I can identify and report safety violations.

The system must detect when two or more aircraft may eventually violate safety rules. Upon detecting a violation, the system should log the event and notify the involved aircraft via signals.
