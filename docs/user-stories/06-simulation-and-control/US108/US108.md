# US108 - Enforce Step-by-Step Simulation Using Semaphores

## User Story

As a simulation engine, I want to enforce step-by-step simulation using semaphores so that all flight processes advance in lockstep.

The parent process must coordinate the execution of flight processes using semaphores. Each flight process should wait for permission to execute the current time step, perform its movement calculation, write its updated state to shared memory, and then notify the parent process that it has completed the step. The parent process must only advance to the next time step after all active flight processes have completed the current one.
