# US109 - Generate Simulation Report

## User Story

As a Flight Control Operator, I want to generate a simulation report so that I can analyze the execution results of the simulated flights.

The report should summarize the simulation execution, including included flights, aircraft position data, safety violation events, flight process status, relevant warnings/errors and the final simulation outcome.
