# US114 - Simulated Flights Visualization

## User Story

As a Flight Control Operator, I want to view the progress of the simulated flights.

The Flights Logging Server application must contain an HTTP server that provides a status page to clients running a standard web browser. The data about the running simulated flights may be presented as a table, but it must be automatically updated to show the step-by-step progress of the simulation. AJAX must be used to update the presented data without reloading the page.
