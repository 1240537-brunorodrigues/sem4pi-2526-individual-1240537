# US112 - Monthly Report Generation

## User Story

As a Flight Control Operator, I want to generate a monthly statistics report.

This will be one of a number of reports, such as compliance reports, incident reports and other future report types, that the system will have to generate. This work should be foundational to guarantee that all reports follow a consistent branding and structure.

Each report type may have a specific way of collecting data, specific sections and graphics.
