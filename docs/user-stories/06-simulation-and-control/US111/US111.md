# US111 - Generate a Simulation Report

## User Story

As a Flight Control Operator, I want to receive a summary of the simulation results so that I can determine if the programmed flights are safe to run.

The system must generate a report and store it in a file. The report should include the total number of flights, their execution status, safety violation timestamps and positions, and whether the scheduled flight plan passed or failed validation.
