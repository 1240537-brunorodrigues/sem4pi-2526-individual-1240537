# US107 - Notify Report Thread on Safety Violation

## User Story

As a Safety Violation Detection Thread, I want to notify the Report Generation Thread when a safety violation occurs, so that the report can include updated violation information.

When a safety violation is detected, the safety detection thread must update the shared violation state and notify the report generation thread using a condition variable. The report generation thread must wait for this notification and process the violation event safely using mutex-protected shared data.
